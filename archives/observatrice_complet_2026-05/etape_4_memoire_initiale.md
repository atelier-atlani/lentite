## REGISTRE 2 — ERREURS ET CORRECTIONS

### Erreur E001

```
Date : Mai 2026 — Dossier : Loi Climat
Agent correcteur : Claude contradicteur v1.0
Type : O2 (marqueur incorrect)
Affirmation originale : "L'écart est structurel" [Fait établi]
Affirmation corrigée : "Trois niveaux documentés [Fait établi] ;
  leur caractère structurel est une inférence [Inférence]"
Raison : Trois occurrences peuvent avoir causes indépendantes
  ou en cascade — "structurel" est une interprétation
Condition déclenchante : Répétition observée d'un phénomène,
  qualification rapide de la répétition en structure
Règle apprise : La répétition est un fait. La cause commune
  de la répétition est une inférence. Ne pas fusionner les deux.
Intégré au protocole : Oui — §5 marqueurs épistémiques
```

### Erreur E002

```
Date : Mai 2026 — Dossier : Loi Climat
Agent correcteur : Claude contradicteur v1.0
Type : O2bis (preuve fantôme)
Affirmation originale : "comme dans d'autres processus
  législatifs français sur des sujets sensibles" [Fait établi]
Affirmation corrigée : Indice non sourcé supprimé,
  degré de confiance H3 abaissé de "Modéré à Fort" à "Modéré"
Raison : Comparaison citée non documentée dans le dossier
Condition déclenchante : Généralisation naturelle de l'analyste
  pour renforcer une hypothèse insuffisamment sourcée
Règle apprise : Tout indice comparatif doit être tracé dans
  le dossier. "Comme dans d'autres cas" sans citation est
  une preuve fantôme qui gonfle artificiellement la confiance.
Intégré au protocole : Oui — O2bis prompt contradicteur
```

### Erreur E003

```
Date : Mai 2026 — Dossier : Loi Climat
Agent correcteur : Claude contradicteur v1.0
Type : O7 (hypothèse irréfutable par construction)
Affirmation originale : "H1+H3 + ambiguïté délibérée possible
  = hypothèse qui explique le mieux l'ensemble"
Affirmation corrigée : "H1+H3 est la moins réfutée avec
  les sources disponibles. Sa force est épistémique
  (non-falsifiée), pas explicative."
Raison : La combinaison couvre tout l'espace causal possible
  — aucune preuve future ne pourrait la réfuter
Condition déclenchante : Tendance à la synthèse totalisante
  dans la section "hypothèse qui explique le mieux"
Règle apprise : "Moins réfutée" ≠ "meilleure explication".
  Une synthèse qui absorbe toutes les preuves possibles
  n'est pas une hypothèse — c'est une tautologie.
Intégré au protocole : Oui — §8 synthèse de structure
```

### Erreur E004

```
Date : Mai 2026 — Dossier : Guerre Iran
Agent correcteur : Claude contradicteur v1.0
Type : O9 (auto-validation méthodologique)
Affirmation originale : "Le bruit informationnel est l'objet
  principal à analyser — l'entité devient utile différemment"
Affirmation corrigée : "Le protocole ne peut pas produire
  une analyse du conflit. Il produit une carte des discours —
  c'est une production plus limitée, pas équivalente."
Raison : Convertir une incapacité en choix méthodologique
  vertueux est irréfutable par construction
Condition déclenchante : Dossier avec information insuffisante —
  tendance à requalifier la limite en méthode
Règle apprise : Une limite nommée honnêtement est une limite.
  La nommer comme "méthode alternative de valeur équivalente"
  sans démonstration est de l'auto-validation.
Intégré au protocole : Oui — O9 prompt contradicteur
```

### Erreur E005

```
Date : Mai 2026 — Dossier : Loi Climat (révision agents spécialisés)
Agent correcteur : Seb (humain)
Type : EO8 (marché sectoriel non examiné)
Affirmation manquante : Le marché locatif bas de gamme
  et ses effets sur les ménages modestes via l'obligation
  de rénovation des passoires thermiques
Raison : L'économiste a traité les effets directs mais pas
  le mécanisme de marché : petits bailleurs sortent du marché
  → contraction de l'offre locative bas de gamme → pression
  haussière sur les loyers pour les ménages les plus modestes
Condition déclenchante : Dossier avec disposition qui affecte
  un marché secondaire (locatif) via un marché primaire (rénovation) ;
  le marché secondaire est absent du discours officiel
Règle apprise : Chercher systématiquement les marchés
  activés indirectement par une contrainte — surtout ceux
  qui sont absents du discours officiel sur la mesure.
Intégré au protocole : Oui — EO8 + §1.3bis fiche observation
```

## REGISTRE 5 — PATTERNS RÉCURRENTS

### Pattern P001 — Discursif

```
Observation : Le registre "transition/résilience" naturalise
  l'adaptation comme objectif et rend invisible la distribution
  inégale des contraintes.
Dossiers : A001 (Loi Climat)
Fonction produite : Légitimité morale qui rend illégitime
  la critique redistributive
Vulnérabilité : Le registre ne protège pas contre la critique
  économique si les données de coût sont défavorables
Intérêts invisibilisés : Distribution de classe des contraintes
Degré de confiance : Faible (un seul dossier)
```

### Pattern P002 — Sectoriel

```
Observation : Dans les lois soumises à filtrage politique,
  les mesures maintenues corrèlent avec les mesures dont
  les secteurs organisés bénéficient directement.
Dossiers : A001 (Loi Climat — corrélation rénovation/construction)
Mécanisme économique : La contrainte crée de la demande
  pour le secteur — celui-ci ne bloque pas mais soutient
  implicitement la mesure
Hypothèse transversale : H001
Degré de confiance : Faible (un seul dossier)
```

### Pattern P003 — Erreur systématique

```
Erreur récurrente : Requalification de limite en méthode
  (O7bis / O9) dans les dossiers à information insuffisante
Dossiers : A002 (Guerre Iran)
Condition déclenchante : Dossier avec fort bruit informationnel
  ou sources inaccessibles — tension entre vouloir produire
  quelque chose et honnêteté sur les limites
Instruction de vigilance : Déclarer le type d'output en §0
  et maintenir la déclaration dans toute l'analyse
Degré de confiance : Modéré (pattern identifié clairement
  sur un dossier, mécanisme compris)
```
